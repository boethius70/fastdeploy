Welcome to FastDeploy!

FastDeploy is a full-featured open source (BSD License) system to automatically deploy operating systems. Or, at least, it aspires to be such a system. It is designed to run automatic O/S deployments over a network using PXE but it does support the building of automated boot CDs and DVDs as well.

There are three main software pieces to run a proper FastDeploy system:
  1. A configurable DHCP server. ISC DHCPv2 or v3; Windows Server DHCP (NT, 2K, 2K3, etc.); and a variety of other DHCP servers. This does not have to run on the same server that is running the FastDeploy web app.
  2. A TFTP server that supports PXE booting. Most newer TFTP server implementations support the PXE boot options. "Known good" TFTP servers include tftpd-hpa and atftpd. Given that most DHCP servers run as the root or superuser this setup script makes no attempt to configure the DHCP, but rather you'll be offered guidance on the few simple options that need to be setup on your ISC or Windows DHCP server.
  3. A web server running PHP4 with either MySQL or PostgreSQL database extensions. This is obviously the key component to run the FastDeploy PHP web application itself.
  4. Either a PostgreSQL or MySQL database. This can be and often is on the same server as the FastDeploy server.

This script will try to determine what piece(s) you do or do not have on this server already. If this script is running obviously you have PHP already but we'll check on the other pieces as well.
