  Fastdeploy : DHCP Server Setup

  Do you want this server to act as a DHCP server? Yes No
Choose whether or not you want this Fastdeploy to act as a DHCP server for this network. DHCP (Dynamic Host Configuration Protocol) is widely used to automatically assign a network configuration to PC workstations and laptops via wired and wireless connections. Please note: Enabling a DHCP server in a local area network (LAN) could affect many machines and take them out of an existing working network configuration. Be careful what piece of the network you are working on when you enable a DHCP server.
  DHCP IP Address Range:
Start of range:... End of range: ...
Enter the IP address range you want to utilize; if, for example, you have a network 10.47.100.0 allocated for DHCP and all IP addresses above .100 available for the purposes of using DHCP please you could set an IP address range starting at 10.47.100.100 and ending at 10.47.100.254. Note that these installer automatically offers a suggested IP range based on the IP address of this server.
  Subnet mask: ...
Enter the subnet mask you want to use for this IP address range. Typically the mask will be something like 255.255.255.0 but your network may be different.
  Gateway: ...
Enter the gateway to be assigned to this DHCP range.
  DNS Server: ...
Enter the DNS server to be assigned to this DHCP range. DNS is used to resolve hostnames to IP addresses.
